HTML Introduction

HTML, short Hypertext Markup Language, serves as backbone of World Wide Web. Developed in the early 1990s by Tim Berners-Lee, HTML is the standard markup language used to create web pages. It provides a structured framework for organizing and formatting content, enabling the seamless display of text, images, multimedia, and interactive elements across various web browsers and devices. HTML utilizes a system of tags to define the structure and presentation of web documents. These tags, enclosed in angle brackets, denote elements such as headings, paragraphs, lists, links, and more. By combining these elements in a hierarchical manner, developers can construct rich and dynamic web pages capable of conveying infromation efecctively to users. HTML works hand in hand with other technologies like CSS and JavaScript to enhance the presentation and functionality of web content. HTML remains an essential skill for anyone looking to create or modify web content. Whether you're a seasoned developer pr just starting your journey in web design, understanding HTML lays foundation for building immersive and engaging online experiences.

HTML Basics

All HTML documents must start with a document type declaration: "!DOCTYPE html". The HTML document itself begins with "html" and ends with "/html". The visible part of the HTML document is between "body" and "/body". The !DOCTYPE declaration represents the document type, and help browsers to display web pages correctly.

HTML Formatting
HTML cotains several elements for defining text with a special meaning.

Example

Formatting elements were designed to display special types of text:
    b - Bold text
    strong - Important text
    i - Italic text
    em - Emphasized text
    mark - Marked text
    small - Smaller text
    del - Deleted text
    ins - Inserted text
    sub - Subscript text
    sup - Superscrpit text
HTML Iframes

An HTML iframe is used to display a web page within a web page.

The HTML iframe tag specifies an inline frame. An inline frame is used to embeded another document within the current HTML document.

HTML Layout

HTML has several semantic elements that define the different parts of a web page:

  • header - Defines a header for a dcument or a section
  • nav - Defines a set of navigation
  • section - Defines in a document
  • article - Defines an independent, self-contained content
  • aside - Defines constent aside from the content(like a sidebar)
  • footer - Defines a footer for a document or a section
  • details - Defines addtional details that user can open and close demand
  • summary - Defines heading for the element

    CSS

    CSS is employed to customized and arrange the visual appearance of HTML documents. It grants control over aspects such as colors, typographhy, spacing and element positioning on a webpage. CSS can be implemented internally within HTML files, externally in dedicated stylesheets files, or directly inline within HTML tags.

    JavaScript

    The HTML script tag is used to define a client-side script(JavaScript). The script element either contains script statements, or it points to an external script file through the src attribute. Common uses for JavaScript are image manipulation, form validation, and dynamic changes content. To select an HTML element, JavaScript most often uses the document.getElementByID() method.
